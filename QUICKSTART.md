# Quick Start Guide

## 1. Create Icons

First, you need to generate the extension icons:

1. Open `icons/create-icons.html` in your browser
2. You'll see 3 canvases on the page
3. **Right-click** on each canvas and select **"Save image as"**
4. Save the files to the `icons` folder with these names:
   - First canvas → `icon16.png`
   - Second canvas → `icon48.png`
   - Third canvas → `icon128.png`

## 2. Install in Chrome

1. Open Chrome
2. Navigate to `chrome://extensions/`
3. Enable **"Developer mode"** in the top right
4. Click **"Load unpacked"**
5. Select the `Highlighter` folder
6. Extension installed! 🎉

## 3. Install in Edge

1. Open Edge
2. Navigate to `edge://extensions/`
3. Enable **"Developer mode"** in the bottom left
4. Click **"Load unpacked"**
5. Select the `Highlighter` folder
6. Extension installed! 🎉

## 4. Make Your First Highlight

1. Go to any website
2. Select some text
3. Right-click → Select **"Highlight (Yellow)"**
4. Text is highlighted! ✨

## 5. Auto Sync (Chrome ↔ Edge)

### Automatic Synchronization

Your highlights **automatically sync** via your Chrome account:

1. **In Chrome:**
   - Sign in with your Google account
   - Use the extension

2. **In Edge:**
   - Sign in with the **same account**
   - Install the extension
   - Your highlights appear automatically!

**No setup required** - your Chrome account handles everything.

### Manual Backup

If you prefer, you can also use manual backup:

1. "Export" → JSON file downloads
2. "Import" → Load the file on another device
3. Refresh the page (F5)

## Keyboard Shortcuts

- `Ctrl+Shift+H` → Yellow
- `Ctrl+Shift+G` → Green
- `Ctrl+Shift+B` → Blue
- `Ctrl+Shift+P` → Pink
- `Ctrl+Shift+R` → Remove Highlight

## Tips

✅ Highlights are automatically saved
✅ Highlights automatically load when you revisit pages
✅ **Ctrl+Click** to remove highlights (easiest method)
✅ View statistics from the popup

**Happy highlighting! 🖍️**
